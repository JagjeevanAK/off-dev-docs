# API Reference

<Callout type="info">
Knowledge panel facets is a simple API that allows users to get knowledge-panels based on different facets.
</Callout>

## Base URL

The API must be prefixed with `/knowledge_panel?`.

The full URL structure:

```json title="Knowledge Panel API Endpoint"
https://facets-kp.openfoodfacts.org/knowledge_panel?facet_tag={desired-facet}&value_tag={facet-value}&lang_code={language}&country={desired-country}
```

## Parameters

<Cards>
<Card title="facet_tag" description="The desired facet to query" />
<Card title="value_tag" description="The specific value for the facet" />
<Card title="lang_code" description="Language code for localization" />
<Card title="country" description="Country code for region-specific data" />
</Cards>
