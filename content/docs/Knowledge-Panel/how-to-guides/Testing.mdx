# Testing

<Callout type="info">
This guide covers testing for both Docker and virtual environment setups.
</Callout>

<Tabs items={['Docker Setup', 'Virtual Environment']}>
<Tab value="Docker Setup">

<Steps>
<Step title="Build Language Dependencies">

```bash title="Build Language Files"
make build_lang
```

</Step>
<Step title="Run Tests">

```bash title="Run Tests with Docker"
docker compose run --rm facets-api pytest tests
```

or

```bash title="Run All Tests"
make all
```

</Step>
</Steps>

</Tab>
<Tab value="Virtual Environment">

<Steps>
<Step title="Build Language Dependencies">

```bash title="Build Language Files"
make build_lang
```

</Step>
<Step title="Check Tests Directory">

Checkout `tests` directory

</Step>
<Step title="Run Tests">

```bash title="Run Tests with Pytest"
pytest 
```

</Step>
</Steps>

</Tab>
</Tabs>

<Callout type="tip">
Make sure all tests pass before submitting your pull request!
</Callout>
